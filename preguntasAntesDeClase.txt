Preguntas antes de la clase
--------------------------------------------------------------------------------
1)¿Cúal es el propósito de este curso? ¿Qué vamos a lograr con esto?
  --> El objetivo del curso es principalmente tener una visión general
      con la intención de formarnos con la idea de que construir un software
      de calidad empresarial, no es complidado.
      Se deben usar prácticas de buena programación, las cuales tendremos como
      referencia para poder construir software más sencillo y reetilizable.
      Dando la posibilidad de que este vivo y que pueda ser reutilizable a
      medida del tiempo.


2)¿Qué tipo de patrones de diseño vamos a ejecutar?
  --> Se verán patrones de arquitectura o de diseño, permitiendo ver el
      problema más desde arriba, utilizando las ventajas de descomponer
      un problema lo que a futuro formará un sistema con todos los
      componentes que lo conforman.


3)¿Qué tipo de lenguaje vamos a utilizar? ¿Qué necesitamos instalar?
  --> Se utilizará Python(3.4) y Javascript. No quiere decir que no se pueda
      usar otros.
  --> Debemos tener un entorno visual que nos permita programar como vim, emacs,
      atom. Para usar Python estaremos usando GUINI.
  --> Se usará ryno, un precompilador de javscript por lado del servidor, lo que
      nos ayudará a correr javscript sin necesidad de un browser.


4)¿Por qué son importantes los patrones de diseño en la programación?
  --> Nos permite ahorrar mucho más tiempo a la hora de ejecutar desarrollo.
      Nos ayudan a estructurar y descomponer el problema a resolver, por ejemplo
      el uso de frameworks a la hora de desarrollar. Tienen incluidos algunos de
      los patrones de diseño, permitiendo despreocuparnos de la estructura arqui-
      tectónica, permitiendo así preocuparnos por el desarrollo directamente.
  --> Esto nos permitirá un mejor y más fácil mantenimiento.
